#  Mobile Sales Data Analytics Dashboard

## Overview
This project analyzes mobile phone sales data to uncover insights about sales performance, customer behavior, and payment preferences. The workflow includes data cleaning, data transformation, and building an interactive Power BI dashboard to visualize key business metrics.

The dashboard helps stakeholders quickly understand sales trends, top-performing products, regional performance, and customer ratings.

---

## Dataset
The dataset contains monthly mobile sales records from different cities in India.

---

Key columns include:
- Date / Month
- City
- Mobile Model
- Quantity Sold
- Sales Amount
- Payment Method (UPI, Debit Card, Credit Card, Cash)
- Customer Rating
- Transaction ID

All monthly datasets were merged and cleaned to create a single dataset for analysis.

---

## Tools & Technologies
- Microsoft Excel – Data storage
- Power BI – Dashboard development
- Power Query – Data cleaning and transformation
- DAX (Data Analysis Expressions) – Calculated measures and metrics

---

## Data Cleaning & Preparation
The following steps were performed before building the dashboard:

- Combined multiple monthly datasets
- Removed duplicate records
- Handled missing values
- Standardized column names and formats
- Converted date fields into proper date format
- Created calculated columns and measures

---

## Dashboard Results
The Power BI dashboard provides several key insights through different visualizations.

Key Metrics:
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Sales Value

Visualizations:
- Total Quantity by Month – Shows monthly sales trends
- Sales by Mobile Model – Identifies top-selling models
- Sales by City (Map) – Displays geographic sales distribution
- Transaction by Payment Method – Shows customer payment preferences
- Customer Rating Distribution – Displays customer satisfaction
- Sales by Day of Week – Identifies peak sales days
